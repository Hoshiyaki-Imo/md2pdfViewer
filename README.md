# ![トプ画](https://raw.githubusercontent.com/Hoshiyaki-Imo/md2pdfViewer/refs/heads/main/icon.ico)md2pdfViewer


## Benefit
- You can convert markdown file to some type of file easily
- This tool is for only markdown file

## Usage
1. Get exe file from release tab
1. Run this file(maybe show alert. please push allow or ok button.)
1. If you want to set some setting, please set from edit tab.
